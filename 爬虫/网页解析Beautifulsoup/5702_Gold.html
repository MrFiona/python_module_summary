WorkAround/BKM\n(mandatory):

BIOS settings:

1).\nFast Cold Boot, Fast Warm Boot need to be disabled. 

     EDKII menu -> Socket Configuration -> Memory Configuration->Fast Boot<disable>,Fast cold Boot<disable>

2). Volatile Memory Mode: 2LM

      EDKII menu -> Socket Configuration -> Memory Configuration -> Memory Map and set Volatile Memory Mode to <2LM>

Pedigree and Board Jumper Settings: 2S Guide

Note: Only mandatory and necessary optional rework is implemented by BKC team

\xc2\xb7      2s

o NC_RWK_I (Mandatory): Update BMC FW

o NC_RWK_K (Mandatory): Avoiding false LAN Wake Activations

o NC_RWK_L (Mandatory): Update CPLD FW

o NC_RWK_M (Mandatory): No-Boot without ITP issue fix

o Note: In case you have implemented Optional rework 20, you should move switch S4E2.4 to OFF position, otherwise the default position is ON.
